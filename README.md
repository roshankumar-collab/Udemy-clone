# Udemy Clone (HTML & CSS)

## 📖 Overview
This project is a simple **Udemy clone website** built entirely with **HTML and CSS**.  
It demonstrates how to design a clean, responsive, and user‑friendly online learning platform interface without using JavaScript or backend technologies.

## ✨ Features
- 🎨 Responsive layout using pure CSS  
- 📚 Course listing page with categories  
- 👤 Instructor and student sections  
- 🖥️ Homepage with banner and featured courses  
- 📱 Mobile‑friendly design  

## 🛠️ Technologies Used
- **HTML5** – for structure and semantic markup  
- **CSS3** – for styling, layout, and responsiveness  

## 🚀 How to Run
1. Clone or download this repository.  
2. Open the project folder.  
3. Run `index.html` in your browser.  
4. Explore the pages and design.
